Hush — Background Noise for Windows
=====================================

RUNNING THE APP
  Double-click hush.exe to start.
  The app runs silently in the system tray (near the clock).

USING THE APP
  Left-click the tray icon   — toggle play / pause
  Double-click the tray icon — open volume slider
  Right-click the tray icon  — menu (noise type, volume, quit)

PIN THE ICON NEXT TO THE CLOCK
  Windows hides new tray icons by default. To keep Hush visible:
  1. Right-click the taskbar and choose "Taskbar settings"
  2. Scroll to "Other system tray icons"
  3. Find "hush" and toggle it ON

AUTO-START WITH WINDOWS (optional)
  1. Press Win+R, type: taskschd.msc and hit Enter
  2. Click "Create Basic Task" in the right panel
  3. Name it "Hush", click Next
  4. Set trigger to "When I log on", click Next
  5. Set action to "Start a program", click Next
  6. Browse to hush.exe, click Next then Finish
  Hush will now start automatically when you log in.
